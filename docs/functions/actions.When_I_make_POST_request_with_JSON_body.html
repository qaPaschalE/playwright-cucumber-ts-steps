    Function When_I_make_POST_request_with_JSON_body

    • When_I_make_POST_request_with_JSON_body(
          this: CustomWorld,
          url: string,
          docString: string,
      ): Promise<void>

      Makes a POST request to the given URL with the provided JSON body and stores the response details.

      When I make a POST request to {string} with JSON body:
      """
      { "username": "user", "password": "pass" }
      """

      Parameters

      • this: CustomWorld
      • url: string

        The URL to which the POST request will be made.

      • docString: string

        A Cucumber DocString containing the JSON payload for the request body.

      Returns Promise<void>

      Example

      Scenario: Login via API
      When I make a POST request to "https://api.example.com/login" with JSON body:
      """
      { "email": "test@example.com", "password": "password123" }
      """
      Then the last response status should be 200
      And the last response body should contain "token"

      Remarks

      The docString must contain valid JSON. The request will have a Content-Type of application/json. The response object (including status and body) is stored in CustomWorld.data.lastResponse | this.data.lastResponse.
